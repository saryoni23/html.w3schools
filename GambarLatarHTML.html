A background image for a div element:

You can specify background images for any visible HTML element. In this example, the background image is specified for a div element. By default, the background-image will repeat itself in the direction(s) where it is smaller than the element where it is specified. Background Cover

Set the background-size property to "cover" and the background image will cover the entire element, in this case the body element.
